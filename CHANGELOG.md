# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-alpha] - 2026-05-15

### Added
- **Firebase Integration:** Fully migrated to cloud storage and authentication. Users can securely log in and have reading data seamlessly synced across devices.
- **Theming Engine:** Introduced a fully dynamic theming system with Light, Dark, and our signature Sepia theme.
- **Dynamic Favicon:** The website's favicon intelligently adapts its color to match the current active theme.
- **Bilingual Support (i18n):** Added a seamless language toggle, allowing users to switch the interface between English (EN) and Indonesian (ID) on the fly.
- **Responsive UI/UX:** Built with Tailwind CSS and enhanced with Framer Motion to provide a smooth, animated, and responsive experience across all screen sizes (mobile, tablet, desktop).
- **Personal Library Dashboard:** Added functionality for users to start adding books, tracking their reading status, and storing personal reflections in an organized layout.

### Changed
- Migrated local state management to Firebase Cloud Firestore for persistence.
